Ehsan Tork: Öyrənmə və paylaşma ehtirasına malik Full-Stack Developer və Veb Dizayner

Mən Açıq Mənbə və Azad Proqram təminatına yönəlmiş proqramçı və proqram təminatı developeriyəm. Sevimli ƏS-im GNU/Linux-dur və əsas distribusiyam Arch Linux-dur. Pacman və onun çevikliyini sevirəm.

Sevimli dilim C/C++-dır və Qt-ni sevirəm. Java yaxşıdır və Java ilə proqramlaşdırmaqdan həzz alıram, lakin C++ mənim üçün Java-dan daha güclüdür. Son illərdə backend inkişafı üçün Rust və yerli desktop tətbiqləri üçün Iced/Slint üzərində fokuslanmışam.

Python-u da sevirəm, lakin eyni zamanda C++ və Python kodları yazmaq istədiyim zaman bir az çaşqınlıq yaradır. Nöqtəli vergüllər səbəbindən! 😄

2008-ci ildən Linux istifadə edirəm. İlk distribusiyam Red Hat 6 idi.
